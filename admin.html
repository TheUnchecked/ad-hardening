Come aggiungere il controllo:

          
1. Copia il JSON qui sopra.

          
2. Apri data/checklist.json su GitHub (icona matita).

          
3. Trova la categoria giusta e incolla l'oggetto nell'array "items", separato da virgola dall'elemento precedente.

          
4. Clicca Commit changes — il sito si aggiorna in pochi secondi.

          
5. Usa la tab Valida JSON per verificare che il file sia corretto prima del commit.
